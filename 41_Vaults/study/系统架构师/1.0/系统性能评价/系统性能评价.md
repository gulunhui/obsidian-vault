# 性能评价指标
主频：时钟频率。晶振体产生的时间脉冲MHz GHz
时钟周期：计算机中最基本的单位，一个时间脉冲的时间，也做震荡周期。
机器周期：CPU周期。一个基本操作所需的时间就是机器周期。一个机器周期由若干的时间周期组成。一个指令周期由若干个机器周期组成
CPI:每条指令执行所用的时钟周期数。是一个平均值
IPC 等于CPI的倒数
MIPS，每秒百万条指令的执行速度
MIPS=指令条数/(执行时间X 1000000)=主频/CPI=主频* X IPC

# 性能评估方法
评价程序
真实程序：评测准确度最高
基准测试程序：最多，最频繁的核心程序。较好方法。
核心程序，具有代表性的，最耗时的程序段汇集。代码很短，但是非常关键
小型基准程序：100行以内。
合成基准程序：人为合成的测试程序。先统计再人为写出测试程序
