# 切克兰德方法
7步骤
1. 认识问题
2. 根底定义
3. 建立概念模型
4. 比较及探寻
5. 选择
6. 设计与实施
7. 评估与反馈

# 并行工程
目标是提高质量、降低成本、缩短开发周期
1. 设计开发期间将概念设计、结构设计、工艺设计、最终需求等结合起来，保证以最快速度按要求完成
2. 各项工作由于此相关的项目小组完成。小组成员各自安排自身工作，但可以随时或定期反馈
3. 依据适当的信息系统工具，反馈与协调整个项目的进行

# 综合集成方法
1. 定性研究与定量研究有机结合
2. 科学理论与经验知识结合
3. 把宏观和微观结合

	3. 需求的可追溯性
	4. 每种表示的事后验证
2. 渐进迭代式开发
	1. 适合需求不清晰、不确定
	2. 较小的不太复杂项目
	3. 灵活性
	4. 通过剪裁突出产品开发的核心活动
3. 精益开发
	1. 动态、知识驱动、以客户为中心
	2. 向客服交付最大价值并减少浪费
